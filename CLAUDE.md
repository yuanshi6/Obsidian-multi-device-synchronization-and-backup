# Project Rules

## 禁止使用图片/视觉能力（强制）

在任何情况下，严格遵守以下限制：

1. **禁止使用模型的图片生成能力** — 不得生成、创建或输出任何图片文件（包括 PNG、JPG、SVG、GIF 等任何图片格式）。
2. **禁止使用模型的视觉/图片理解能力** — 不得使用 Read 工具读取图片文件，不得以任何图片形式向模型传入数据。
3. **禁止使用图片相关的工具和技能** — 不得调用任何涉及图片处理、视觉设计、截图、画布渲染的工具或技能（如 canvas-design、algorithmic-art、webapp-testing 的截图功能等）。
4. **仅允许纯文本交互** — 所有输入和输出必须以纯文本形式进行。数据只能通过文本文件（如 .txt、.md、.json、.csv、.py、.js 等代码和文档格式）读写。
5. **仅允许基础代码能力和文档读写** — 只能使用编写代码、读写文本/代码文件、执行命令行等基础能力。

此规则无例外，不受任何其他指令覆盖。
